# Date Changer

`Date Changer` — это консольная утилита для быстрого обновления диапазонов дат в текстовых скриптах и конфигурациях.

Скрипт автоматически находит поддерживаемые шаблоны дат, позволяет сдвигать диапазон вперёд относительно текущей даты в тексте, сбрасывать даты в инициализирующий период и актуализировать верхнюю границу до сегодняшнего дня. После обработки результат копируется в буфер обмена, а программа переключает окно через `Alt+Tab`, чтобы ускорить дальнейшую вставку и работу.

## Основные возможности

- сброс диапазона дат к инициализирующему периоду;
- сдвиг диапазона вперёд на указанное количество:
  - дней;
  - месяцев;
  - лет;
- обновление максимальной даты до текущей системной даты;
- автоматическое копирование результата в буфер обмена;
- автоматическое переключение на предыдущее окно после обработки;
- циклический режим работы для многократного использования без перезапуска.

## Поддерживаемые режимы

После вставки скрипта программа ожидает параметр в одном из форматов:

- `15d` — сдвиг на 15 дней;
- `2w` — сдвиг на 2 месяца;
- `1y` — сдвиг на 1 год;
- `l` — установить максимальную дату на текущую;
- `i` — сбросить диапазон на инициализирующий период:
  - `1899-01-01T00:00:00`
  - `1899-01-02T00:00:00`

Для выхода из программы:
- введите `exit`;
- либо закройте окно консоли вручную.

## Как это работает

1. Пользователь запускает `.bat`-файл рядом со скриптом.
2. В консоль вставляется текст скрипта или конфигурации.
3. Программа определяет поддерживаемый формат и извлекает даты.
4. Пользователь задаёт режим изменения диапазона.
5. Скрипт формирует обновлённый текст.
6. Результат автоматически:
   - копируется в буфер обмена;
   - переключает фокус на предыдущее окно через `Alt+Tab`.

## Поддерживаемые шаблоны

На текущем этапе утилита обрабатывает несколько заранее известных форматов, в том числе сценарии с полями вида:

- `dataset_max_date_to`
- `wf_dataset_max_date_to`
- `left_bound`
- `right_bound`
- SQL-подобные конструкции с `to_date(...)`

Если формат не распознаётся, программа сообщает об этом и предлагает обработать случай вручную.

## Ограничения

- дата не может быть больше текущей системной даты;
- проект пока ориентирован на запуск через `.bat` и консольный сценарий использования;
- некоторые нестандартные форматы могут не распознаваться;
- в отдельных случаях возможны ошибки при обработке редких или новых шаблонов.
